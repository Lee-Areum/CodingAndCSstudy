## 삽입정렬(Insertion Sort)

삽입정렬은 선택정렬과 비슷하지만 삽입정렬이 더 효율적이다.

i = 1~N으로 i의 값을 i 이전의 값들과 비교해서 i의 값이 들어가야 하는 위치에 삽입하는 알고리즘 이다.

ex) 1432

i=1 ⇒ 1432

i=2 ⇒ 1342

i=3 ⇒ 1234

### 시간복잡도

- 최선의 경우 O(n) / 최악 or 평균 : O(n^2)

### 공간복잡도

- O(n)

### 장점

- 단순한 알고리즘
- 다른 메모리 공간이 필요하지 않음
- 이미 정렬된 경우에는 매우 효율적

### 단점

- 비효율적
